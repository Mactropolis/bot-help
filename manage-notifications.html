📜 Manage Notifications
← Voltar
Available Operations
🔍 Search
Filtra as opções do menu com base em palavras-chave específicas.
📝 Edit
Modifica os detalhes da notificação.
⚙️ Notification is active/inactive
Alterna entre ativar ou desativar a notificação.
Clique para alternar entre habilitar ou desabilitar.
Ao ativar uma notificação sem repetição, o horário será mantido, mas a data será alterada para hoje ou para amanhã, caso o horário já tenha passado.
👀 Preview
Permite ver como a notificação ficará quando for enviada.
🗑️ Message Cleanup
Configura um tempo personalizado para exclusão da mensagem desta notificação.
📝 Change Channel
⚠️ Atualiza o canal desta notificação quando o canal atual não estiver disponível.
🗑️ Delete
Remove a notificação selecionada.
