📝 Mark Attendance
← Voltar
Neste menu você pode criar uma nova sessão de presença ou selecionar uma sessão existente para incluir, excluir ou modificar a presença dos jogadores.
Session Selection
🆕 New Session
Criar um novo evento de presença.
📂 Select Session
Selecionar um evento existente para incluir, excluir ou modificar a presença dos jogadores.
Player Selection
No próximo menu será exibida uma lista dos jogadores da aliança.
Vários jogadores podem ser selecionados em cada página da lista.
Caso o objetivo seja informar a pontuação, cada jogador deve ser selecionado individualmente.
Attendance Status
Após a seleção, deve ser informado se o jogador está:
  • Present
  • Absent
Se Present for informado e apenas um jogador tiver sido selecionado, você pode informar a pontuação individual desse jogador.
Se vários jogadores forem selecionados, você só poderá informar uma pontuação geral para todos.
Finish
Ao finalizar, clique em Finish Attendance para salvar as alterações.
