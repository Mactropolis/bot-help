O Setup Wizard ajuda você a configurar rapidamente notificações de eventos.
No primeiro menu, Global Settings, você define as configurações gerais.
Global Settings
📍 Channel: informe o canal onde as notificações serão enviadas. Geralmente utilizamos o canal alerts.
📢 Mention: escolha como os usuários serão mencionados:
  • 1) @everyone (notifica todos)
  • 2) Um cargo específico
  • 3) Um usuário específico
  • 4) Não mencionar ninguém
Dica: dê preferência à opção 4 para evitar spam.
🕰️ Notification Times: define quantas vezes e com quanto tempo de antecedência a notificação será enviada.
🌍 Timezone: permite definir um fuso horário específico.
Continue to Event Selection
Configure Events
Neste menu, você deve selecionar ou desselecionar os eventos que deseja notificar.
Clique em um evento para configurá-lo. Eventos configurados exibem ✅
Clique novamente em um evento já configurado para removê-lo.
