📢 Notification System
← Voltar
O Sistema de Notificações pode ser utilizado para criar alertas que avisam os jogadores sobre eventos futuros. Ele é totalmente personalizável e pode ser usado para qualquer tipo de evento. Utilize as opções abaixo para começar.
Operações disponíveis
🧙 Setup Wizard →
Configuração rápida e fácil para notificações de eventos mais comuns
O assistente irá guiá-lo passo a passo durante todo o processo
Execute novamente o assistente em um canal para atualizar notificações existentes
⏰ Custom Notification →
Configure uma nova notificação com horário, mensagem, menções e repetição personalizados
Suporta mensagens simples e embeds avançados
Ideal para eventos não cobertos pelo assistente
📜 Manage Notifications →
Visualize todas as notificações existentes
Edite, ative/desative ou exclua notificações
📅 Schedule Boards →
Crie painéis de agenda que exibem notificações futuras
Atualização automática ao criar, editar ou excluir notificações
Suporte para painéis globais ou por canal com configurações personalizadas
📄 Event Templates →
Navegue por modelos de eventos pré-configurados
Visualize e modifique os padrões de notificação
⚙️ Settings →
Configure se as notificações enviadas serão removidas automaticamente
Defina o tempo padrão para exclusão das notificações
